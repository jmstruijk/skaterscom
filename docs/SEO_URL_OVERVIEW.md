## ğŸ¯ URL Structure Best Practices

### âœ… What We're Doing Right:

1. **Clean, Readable URLs**
   - `/skate-parks/near-me` (not `/sp-nm`)
   - `/locations/ca/los-angeles` (not `/loc?s=ca&c=la`)

2. **Keyword-Rich URLs**
   - URLs contain target keywords
   - Hyphens separate words (not underscores)

3. **Logical Hierarchy**
   - `/locations/{state}/{city}` (clear structure)
   - `/skate-parks/{state}/{city}` (sport-specific)

4. **Consistent Patterns**
   - All sport-city pages follow same pattern
   - All venue slugs include city and state

5. **SEO-Friendly Slugs**
   - Venue slugs: `{name}-{city}-{state}`
   - City slugs: lowercase with hyphens
